To learn more about the Eclipse Process Framework (EPF), read Per Kroll's article, The Eclipse Process Framework project (IBM® developerWorks®, November 2005)

  • IBM Rational Method Composer: Part 1: Key concepts by Peter Haumer (IBM developerWorks, December 2005)
  • EPF Composer Overview, Part 1 and Part 2 .  (Get Adobe Reader)
  • For an in-depth explanation of the metamodel on which EPF Composer is based, see OMG: Software Process Engineering Meta-Model, Version 2.0, 2008.
