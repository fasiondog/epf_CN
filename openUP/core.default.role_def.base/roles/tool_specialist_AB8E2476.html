角色：工具支持专员
工具支持专员（Tools Specilist）提供项目使用的工具，并为其提供技术支持。包括选择和获取工具、配置和设置它们，以及验证它们可以工作。

    技能

    扮演此角色的个人需要有一系列广泛的技能。包括理解项目所使用的低层过程，以及在项目启动时进行培训。在为项目比较和选择工具时，通用的系统分析技能是有益的。关于开发平台的知识是必需的，尤其是关于网络的问题。承担该角色的人员还需要良好的沟通过技能以及一个服务的心态，因为他或她需要在支持中接触项目成员，如在安装和其他工具故障时。

    分配方法

    此角色可以通过如下方式进行分配：

    • 分配一位或多位员工同时执行此角色和开发人员角色。这是通常被采用的方式，尤其是小到中等规模的团队。
    • 指派专人执行此角色。这是通常被采用的方式，并且特别适合于大型的团队或者略小的团队，在那里有效的工具支持和流程自动化是项目开发计划的一个关键方面。
