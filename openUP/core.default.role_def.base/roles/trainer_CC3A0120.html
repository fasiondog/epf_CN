角色：培训师
培训师（Trainer ）为系统最终用户以及产品维护支持人员提供培训。
角色集：部署实施
关系
主要描述

许多组织雇佣培训师为不同的受众交付产品培训。与软件开发人员不同，大多数培训师在交付培训教材方面有着丰富的经验，以合适的方式向最终用户沟通复杂的概念。他们带来必要的创造力和才华，与开发团队的实现和测试技能形成了互补。

人员配备
技能

一个培训师应该具备以下技能：

  • 对正开发的特性所在领域有深刻的理解
  • 为最终用户和应用支持人员提供有效的培训、实验、以及研讨材料
  • 坚持企业培训标准
