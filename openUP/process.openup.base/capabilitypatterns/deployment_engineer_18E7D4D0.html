角色描述符：部署工程师
部署工程师（Deployment Engineer）负责将一个或多个发布版本安全的部署到生产环境。
基于方法角色：部署工程师
关系
执行:
  • 策划部署
负责:
  • Backout Plan
  • Release
  • Release Communications
  • 部署计划
其他执行:修改:
  • 部署计划
主要描述

部署工程师协作部署经理。此团队角色对生产环境的安全至关重要，帮助阻止在生产环境中引入坏的或未经测试的代码。部署工程师通过完成任务来支持部署经理，持续的领导、促使并协调同步的版本，通过使用流程程序来最大化交付给客户的价值。
