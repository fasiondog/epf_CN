角色描述符：项目经理
项目经理（Project Manager）领导项目策划，协调与利益相关人的互动，以及使项目团队聚焦于满足项目目标。
基于方法角色：项目经理
关系
执行:
  • Assess Results
  • Manage Iteration
  • Plan Iteration
负责:
  • Iteration Plan
  • Risk List
  • Work Items List
  • 项目计划
其他执行:修改:
  • Iteration Plan
  • Risk List
  • Work Items List
主要描述

该角色：

  • 指导团队推动项目获得成功，以及客户对产品的接受。
  • 负责项目的输出和客户对产品的接受。
  • 负责评估项目风险，并通过缓解措施控制这些风险。
  • 运用管理知识、技能、工具和技术以及时交付特定项目期望的结果
